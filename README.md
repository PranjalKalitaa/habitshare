# 🏆 HabitShare

Build a habit streak. Stay accountable. Share with friends.

HabitShare is a premium, minimal habit-tracking application designed to help you build consistency and stay motivated through streaks, countdown reminders, and beautiful golden premium features.

---

## 🚀 Get Started

### 📱 1. Install Android App (APK)
You can download and install the official Android App directly on your mobile device:

*   👉 **[Download HabitShare APK (habitshare.apk)](https://github.com/PranjalKalitaa/habitshare/raw/main/habitshare.apk)**

> 💡 *Note: Since this APK is self-signed, when you open the downloaded file your phone may show a "Block by Play Protect" or "Unknown Sources" warning. Simply click **"Install Anyway"** to complete the installation.*

### 🌐 2. Use the Web App (PWA)
No install needed! Open the app instantly in any browser:
*   👉 **[https://habit-share-app.web.app](https://habit-share-app.web.app)**

## ✨ Features

*   **🔒 Multi-Habit Limits (Free vs. Premium)**
    *   *Free Plan:* Limited to 1 active habit.
    *   *Premium Plan:* Track up to 3 habits simultaneously.
*   **✨ Golden Premium Account Overhaul**
    *   Sleek dark gold aesthetics with glassmorphic dashboard designs.
    *   Custom avatar uploading and editable display name synced to your profile.
*   **⏰ Check-In Timer & Reminders**
    *   Interactive countdown timer below your streak ring.
    *   Automatic daily notifications to keep you on track before your streak expires.
*   **💳 Stripe Payment Integration**
    *   Fully integrated subscription flow (Monthly / Yearly) using Stripe Checkout.
*   **📡 Firebase Database Backend**
    *   Instant email-verification gateway and secure user accounts.
